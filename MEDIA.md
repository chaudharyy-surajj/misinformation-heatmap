## 🎬 How to Add Media

### Adding Screenshots
1. Take screenshots of the application
2. Save them in `docs/images/` directory
3. Update the markdown image links above
4. Use descriptive filenames (e.g., `dashboard-overview.png`)

### Adding Videos
1. Upload videos to YouTube, Vimeo, or similar platform
2. Update the video links in the Video Demonstrations section
3. Add thumbnail images for better preview
4. Consider creating GIFs for quick demonstrations

### Recommended Tools
- **Screenshots**: Windows Snipping Tool, ShareX, Greenshot
- **Screen Recording**: OBS Studio, ShareX, Windows Game Bar
- **GIF Creation**: ScreenToGif, Gifski
- **Image Optimization**: TinyPNG, ImageOptim

## 📝 Media Guidelines

### Screenshot Requirements
- **Resolution**: Minimum 1920x1080 for desktop views
- **Format**: PNG for UI screenshots, JPG for photos
- **File Size**: Optimize to < 500KB per image
- **Content**: Remove any sensitive/personal information

### Video Requirements
- **Resolution**: 1080p or higher
- **Format**: MP4 (H.264 codec)
- **Duration**: 2-5 minutes for demos, < 30 seconds for features
- **Audio**: Clear narration or captions
- **Framerate**: 30fps minimum

### Naming Convention
```
[component]-[feature]-[view].extension

Examples:
- dashboard-overview-desktop.png
- heatmap-interaction-demo.mp4
- api-analysis-request.gif
- mobile-responsive-view.png
```
